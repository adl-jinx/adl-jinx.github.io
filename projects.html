- AI Story Mode: where GPT3.5/4 generate short stories, and the last is the

iGram

interface clone

2023

This project is an indept analysis based on the book No Filter: The Inside Story of Instagram, the book describe how the application shaped our life and how perfect its design is. Every major corporation although has its own way to create UI and UX, as an iOS Developer Student I wanted to rework instagram using Apple's style and guidelines. This clone has no backend and it was a case study to learn about UX, UI and read a great book.
If you're into tech check the book it was suggested to me by a great UX designer and mentor Giselle Katics. landing_page
